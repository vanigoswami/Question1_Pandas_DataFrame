
# Student Data Analysis with Pandas

This project demonstrates core data manipulation tasks using the **Pandas** library in Python. It covers creating a dataset from scratch, handling file I/O (CSV), and performing initial data exploration.

## 📋 Tasks Completed

* [x] Create a DataFrame from raw student data.
* [x] Export the data to a CSV file (`students.csv`).
* [x] Reload the CSV file into a new DataFrame.
* [x] Perform data inspection using `.head()`, `.describe()`, and `.info()`.
* [x] Verify data integrity and types.

## 🛠️ Technologies Used

* **Python 3.x**
* **Pandas Library**
* **Google Colab** (Development Environment)

## 📁 Repository Structure

* `Question1_Pandas_DataFrame.ipynb`: The Jupyter Notebook containing the code and explanations.
* `students.csv`: The exported data file generated by the script.

## 🚀 How to Run

1. Open the `.ipynb` file directly in GitHub to view the code and outputs.
2. To run the code yourself, upload the notebook to [Google Colab](https://colab.research.google.com/).
